# 📱 Contacts App (iOS - API Based)

A modern iOS application to manage contacts using REST APIs.  
This app allows users to fetch, add, update, and delete contacts from a remote server.

---

## 🚀 Features

- 🌐 Fetch contacts from API
- 👤 Add new contact via API
- ✏️ Update contact details
- 🗑 Delete contact from server
- 🔍 Search contacts
- 🔄 Pull-to-refresh support
- ⚡ Fast and responsive UI

---

## 🛠 Tech Stack

- Language: Swift
- Framework: SwiftUI
- Architecture: MVVM
- Networking: URLSession
- API Type: REST API (JSON)
- Minimum iOS Version: iOS 13+

---

## 🌐 API Integration

The app communicates with a backend server using RESTful APIs.

### Example Endpoints:

- GET /contacts → Fetch all contacts
- POST /contacts → Add new contact
- PUT /contacts/{id} → Update contact
- DELETE /contacts/{id} → Delete contact
